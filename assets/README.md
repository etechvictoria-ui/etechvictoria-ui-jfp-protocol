# JFP Assets

This folder contains visual materials related to JFP and related prototype systems.

Assets are provided for visual-reference, educational, and promotional purposes only.

They should not be interpreted as current protocol documentation, verified technical specifications, production claims, financial valuations, security certifications, or operational guarantees.

## Categories

- `wallpapers/` — visual branding and promotional wallpapers
- `diagrams/` — technical diagrams and workflow illustrations
- `screenshots/` — selected interface screenshots and visual references

## Important note

Some visuals may contain historical version numbers or concept-stage terminology.

When that happens, the image should be treated as concept artwork or legacy visual material, not verified documentation.
