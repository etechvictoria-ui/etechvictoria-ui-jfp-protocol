# JFP Examples

This folder will contain practical examples of how to use JFP in real workflows.

## Purpose

Examples demonstrate how to go from a `.jfp` specification to structured AI-assisted execution.

They are different from case studies:

- **examples/** → small, reproducible demos you can run locally
- **case-studies/** → architectural descriptions of larger systems (no source code)

## What will be here

Planned example types:

- minimal build flows
- patch/update flows
- validation-first workflows
- CLI-driven workflows
- multi-step controlled execution scenarios

## How to use

Each example should include:

- a `.jfp` file
- expected CLI output
- explanation of what happens step-by-step

## Status

Early placeholder.

Examples will be added as the JFP tooling and workflow patterns mature.
